QuickTranslate Privacy Policy

QuickTranslate 隐私政策

Effective date: July 20, 2026

生效日期：2026 年 7 月 20 日

QuickTranslate sends only text that you explicitly select or enter to your chosen translation provider when you request a translation. QuickTranslate does not operate a translation proxy server.

1. Data we process

  • Webpage, PDF, or manually entered text that you explicitly submit for translation.
  • Third-party model API keys and your provider, model, language, and interface settings.
  • Extension state, reading state, PDF highlights, and comments stored locally.

2. How data is used

Translation text is used only to complete the translation you request. API keys are used only to authenticate with your chosen provider, refresh the models available to that account, and submit translation requests. PDF highlights, comments, and reading state are used only for local PDF reading and annotation features.

3. Third-party providers

Depending on your selection, translation text and credentials required to complete the request may be sent directly to LibreTranslate, MyMemory, OpenAI, Anthropic, Google Gemini, DeepSeek, xAI, Groq, Mistral, OpenRouter, Alibaba Cloud, Moonshot, Zhipu, or Together. Each provider processes requests under its own terms and privacy policy. Do not submit sensitive content that you do not want your selected provider to process.

4. Local storage and retention

Settings and API keys are stored in the current Chrome profile using chrome.storage.local; Chrome Sync is not used. QuickTranslate does not add encryption to locally stored API keys. Removing the extension or clearing its storage deletes these settings. PDF files, annotations, and reading sessions are stored in the local browser or in PDF files that you save.

5. What we do not do

  • We do not collect or sell complete browsing history.
  • We do not use user data for personalized advertising, credit assessment, or data brokerage.
  • We do not allow developers to manually read text submitted for translation.
  • We do not call translation or model-list services before you confirm the data disclosure.

6. Security

Requests to supported providers use HTTPS. The extension does not execute code returned by providers; responses are handled only as translation text or model metadata. Protect your Chrome profile and API keys.

7. User controls

You can disable translation, delete API keys, withdraw consent to data processing, clear local extension data, or uninstall the extension. Translation and model-list requests are blocked after consent is withdrawn.

8. Donations

The optional support page displays Alipay, WeChat Pay, and PayPal payment destinations. QuickTranslate does not process payments, receive payment details, or store donation transaction information.

9. Limited Use

QuickTranslate's use and transfer of information received from Chrome extension permissions complies with the Chrome Web Store User Data Policy, including the Limited Use requirements. Data is used only to provide translation and related PDF reading features that the user explicitly requests.

10. Policy updates

If our data handling changes materially, we will update this policy, provide a prominent notice in the extension, and obtain consent again before the new processing begins.

11. Contact

Contact the publisher through the support channel listed on the QuickTranslate Chrome Web Store listing.

QuickTranslate 仅在用户主动请求翻译时，将用户明确选择或输入的文本发送给用户选择的翻译服务商。QuickTranslate 不运营翻译中转服务器。

1. 我们处理的数据

  • 用户主动选择或输入并请求翻译的网页、PDF 或手动输入文本。
  • 用户提供的第三方模型 API key、服务商、模型、语言和界面设置。
  • 保存在本机的扩展开关、阅读状态、PDF 高亮和评论。

2. 数据如何使用

翻译文本仅用于完成用户请求的翻译。API key 仅用于认证用户选择的服务商、刷新该账号可用模型及提交翻译请求。PDF 高亮、评论和阅读状态仅用于本地 PDF 阅读与批注功能。

3. 第三方服务商

根据用户的选择，翻译文本及完成请求所需的凭据可能直接发送至 LibreTranslate、MyMemory、OpenAI、Anthropic、Google Gemini、DeepSeek、xAI、Groq、Mistral、OpenRouter、阿里云百炼、Moonshot、智谱或 Together。服务商按照各自的条款和隐私政策处理请求。用户应避免翻译不希望交由所选服务商处理的敏感内容。

4. 本地存储与保留

设置和 API key 使用 chrome.storage.local 保存在当前 Chrome 配置文件中，不使用 Chrome Sync。QuickTranslate 不对本地 API key 额外加密。移除扩展或清除扩展存储会删除这些设置。PDF 文件、批注和阅读会话保存在本机浏览器存储或用户保存的 PDF 文件中。

5. 我们不会做的事情

  • 不会收集或出售完整浏览历史。
  • 不会将用户数据用于个性化广告、信用评估或数据经纪。
  • 不会允许开发者人工阅读用户提交的翻译文本。
  • 不会在用户未确认数据说明时调用翻译或模型列表服务。

6. 安全

与支持的服务商之间的请求使用 HTTPS。扩展不执行服务商返回的代码，只将响应作为翻译文本或模型元数据处理。用户应保护自己的 Chrome 配置文件和 API key。

7. 用户控制

用户可以关闭翻译功能、删除 API key、撤回数据处理同意、清除扩展本地数据，或卸载扩展。撤回同意后，翻译和模型列表请求将被阻止。

8. 捐赠

自愿支持页面会显示支付宝、微信支付和 PayPal 收款方式。QuickTranslate 不处理付款、不接收支付资料，也不保存任何捐赠交易信息。

9. 有限使用

QuickTranslate 对通过 Chrome 扩展权限获得的信息的使用和传输遵守 Chrome Web Store User Data Policy，包括 Limited Use 要求；数据仅用于提供用户明确请求的翻译和相关 PDF 阅读功能。

10. 政策更新

如果数据处理方式发生实质变化，我们会更新本政策，并在扩展内显著提示用户，在新处理开始前重新取得同意。

11. 联系方式

联系发布者：请使用 QuickTranslate Chrome Web Store 商品页面中列出的支持渠道。
